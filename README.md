# ro-device-analysis

RO装置運転データの時系列解析  
― 透過水量が透過水質に与える影響の検証 ―

---

## 🏥 背景

透析治療では高純度の透析液を作成するためにRO水（逆浸透水）が必要である。  
RO水の生成には高価なRO膜が使用され、メーカー推奨では2〜3年ごとの定期交換が必要とされている。

当院では通常 **30 L/min** の流量で運転しているが、  
必要最低限である **25 L/min** に下げることで膜負荷を軽減し、寿命延長につながる可能性があると考えた。

しかし、

> 「流量を下げると水質は悪化しないか？」

という懸念がある。

本研究では、日常運転データを用いて  
**透過水量が透過水質に与える時間的影響を統計的に検証した。**

※ 公開版では守秘義務の観点からダミーデータを使用しています。

---

## 🎯 目的

1. 水量 → 水質 の因果方向を検証する  
2. 即時効果および時間遅れ効果を評価する  
3. 原水温度・季節要因を調整しても関係が残るか確認する  

---

## 📊 データ概要

- 観測期間：約4,300日
- 主な変数：
  - 1段目透過水量（L/min）
  - 1段目透過水質（mS/m）
  - 原水温度（℃）
  - 圧力関連指標
  - 年周期（sin / cos 変換）

---

## 🔬 解析手法

### 1. 相関分析
水量と水質の基本的関係を確認

### 2. ラグ付き回帰分析（OLS）
以下のラグを検証：

- 当日
- 1日遅れ
- 7日遅れ
- 14日遅れ

### 3. 季節調整

- 原水温度
- 年周期（sin / cos）

### 4. HAC（Newey-West）標準誤差

時系列データでは誤差に自己相関が生じる可能性があるため、

```python
cov_type='HAC', cov_kwds={'maxlags':14}
```

を使用し、自己相関および不均一分散に対して頑健な推定を実施。

### 5. グレンジャー因果検定

- 水量 → 水質
- 水質 → 水量（逆方向）

因果の主方向を検証。

---

## 📈 主な結果

### 1. 水量の即時効果

- coef = -0.0015

- p = 0.011（HAC補正後も有意）

→ 水量が低下すると同日に水質は悪化する

### 2. 1日遅れ効果

- coef = -0.0020

- p = 0.003

→ 前日の水量低下も翌日の水質悪化に影響

### 3. 7日・14日遅れ

- 有意ではない

→ 長期持続効果は明確ではない

### 4. 原水温度の影響

- coef = 0.004

- p < 0.001

→ 水質変動の主要因は温度

### 5. 季節要因

sin_year / cos_year ともに有意

→ 温度のみでは説明できない年周期要因が存在

### 6. グレンジャー因果検定

- 水量 → 水質：連続的に有意

- 水質 → 水量：断続的かつ限定的

→ 主方向は「水量 → 水質」である可能性が高い

---

## 🧠 解釈

水量は水質に即時および1日遅れで影響を与える影響量は小さい、水質変動の主因は原水温度および季節要因長期持続的な影響は限定的。
